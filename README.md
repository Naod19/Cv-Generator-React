# CV-Generator
A React Application for Generating A Cv using information from the user and generate a PDF file that can be downloaded.

## Features

- Form Inputs to type in the information
- Navigation buttons to go back and forth between information sections
- Preview the Cv before downloading
- An edit button to edit information

## Built With

- React 
- JavaScript
- CSS
- Vite

## What I learned

- Accessing nested object dat from a state
- Creating a PDF for downloading
